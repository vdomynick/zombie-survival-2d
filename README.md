# Zombie-Survival-2d
Projeto acadêmico da disciplina Desenvolvimentos de Jogos.

## Sobre o projeto 
Zombie Survival 2D é um jogo de sobrevivência em arena 2D em GameMaker para Windows.

## Tecnologias Utilizadas
- GameMaker
- GML (linguagem do GameMaker)
- Github
